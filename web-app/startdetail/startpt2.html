Commands are function names that you submit along with records, to indicate how to use or update that record. Special character codes and sequences that you enter, or that exist in records, are in records as having special meaning. Use of special commands and syntax is optional and will not apply to all records. Commands and syntax are also menu-selectable.
Tags and Categories are an index into activities such as: find information, deep-link to the web, process email, open a folder on your local machine. Tags and categories can be re-used across records to save keystrokes.

UsToDo.com is a text and forms database with simple but powerful touch-points to help you to clip, categorize, prioritize, track, browse, and go to all of your electronic artifacts: accounts, data, documents, emails, events, folders, links, numbers, notes, movies, steps, stickies, thoughts, websearch, etc. An integration point and workflow minimization tool for your web and computer interactions.   

The system has several pieces: 1) a web app, 2) a browser plugin, 3) a smartphone app, and 4) email & text messaging for input and reminders. (Items 1 and 4 are currently implemented in prototype.) Use text records (and largely text-keyboard-based interactions) to save tidbits, manage personal information and calendars, launch apps and links, maintain search/browse history, and help organize and process email (email management is one of the primary applications in UsToDo).     Schedule. Organize. Message. Execute.    

UsToDo records are, at root, simply time-stamped text. Records can be queried with full substring search. Records usually include tags and categories as an index, and contain clickable links to web sites, to open/run local computer folders-files-commands, or to individual (e.g., categorized) email messages. Users can structure text records in any way and use regular expressions to find records of particular structure or value. Tag and category management tools... JSON document format is supported enabling structured database features, forms, and document database query including topic and regular expression filters.

A customizable set of Commands can be embedded in the text (or selected from a menu) to invoke specific UsToDo actions such as: 'QuickLink' to external a web page and 'QuickSearch' into external apps like Amazon, Bing, Email, Ebay, Evernote, Facebook, Google, Maps, Netflix, Weather, Yelp, Zappos, etc. Share or publish associated records to individuals or groups. A set of records can be exported and re-imported after editing with more traditional document editors such as Word or Google Docs.
